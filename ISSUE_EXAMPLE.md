# Issue example

## Заголовок

Запретить пустое имя исполнителя

## Описание

В проекте «Трекер командных задач» нужно добавить проверку пользовательского ввода и защиту от некорректных данных.
Это позволит сделать приложение надежнее и показать работу с GitHub Issues.

## Как закрыть Issue коммитом

```bash
git commit -m "Запретить пустое имя исполнителя. Closes #1"
```

После пуша ветки на GitHub задача закроется автоматически, если номер Issue действительно равен `#1`.
