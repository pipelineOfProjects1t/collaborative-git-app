# Информация о шаблоне

**Номер проекта:** 50

**Название папки:** `collaborative-git-app`

**Шаблон проекта:** `Трекер командных задач`

**Ссылка на GitHub:** `https://github.com/pipelineOfProjects1t/collaborative-git-app`

**Файл данных:** `team_tasks.json`

## Что лежит в папке

- `main.py` — готовое консольное приложение на Python.
- `team_tasks.json` — JSON-файл для хранения данных.
- `README.md` — описание проекта, запуск, ссылка на GitHub и порядок сдачи.
- `GIT_STEPS.md` — пошаговые команды Git и GitHub.
- `ISSUE_EXAMPLE.md` — пример задачи в GitHub Issues.
- `PULL_REQUEST_DESCRIPTION.md` — пример описания Pull Request.
- `tests/` — модульные тесты.

## Какой это шаблон

Это шаблон проекта «Трекер командных задач». Его можно использовать как отдельный репозиторий `collaborative-git-app` для практического кейса по совместной разработке с Git и GitHub.
