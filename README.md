## Описание

Это готовое решение для практического кейса по совместной разработке с Git и GitHub.
Проект представляет собой консольное приложение для имитации командной работы над задачами, исполнителями и статусами.
Приложение написано на Python и работает через консольное меню.
Данные хранятся в файле `team_tasks.json`.

## Меню приложения

1. Добавить запись
2. Показать все записи
3. Показать статистику
4. Удалить запись
5. Выход

## Поля записи

- `task` — Задача
- `assignee` — Исполнитель
- `deadline` — Дедлайн
- `status` — Статус

## Как запустить

```bash
python main.py
```

## Как запустить тесты

```bash
python -m unittest discover
```

## Ветки для разработки

- `feature/add-record` — добавление записи
- `feature/list-and-stats` — просмотр списка и статистика
- `feature/delete-record` — удаление записи

## Пример Issue

**Запретить пустое имя исполнителя**

Описание: нужно улучшить приложение и проверить ввод данных, чтобы пользователь не мог добавить некорректную запись.
