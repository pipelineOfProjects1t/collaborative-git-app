# Book App Pull Request

Репозиторий проекта: https://github.com/pipelineOfProjects1t/book-app-pull-request

## Описание

Это консольное приложение на Python для учёта прочитанных книг. Проект создан как практическое решение кейса по совместной разработке с Git и GitHub. Приложение позволяет добавлять книги, смотреть список, считать среднюю оценку, выводить статистику по авторам и удалять записи. Данные хранятся в файле `books.json`.

## Функции приложения

1. Добавить книгу
2. Показать все книги
3. Показать среднюю оценку
4. Статистика по авторам
5. Удалить книгу
6. Выход

## Пример Issue

Заголовок: Добавить проверку на дубликаты книг

Описание: Пользователь не должен добавлять одну и ту же книгу дважды, если совпадают автор и название.

Коммит для закрытия задачи:

```bash
git commit -m "Добавить проверку дубликатов книг. Closes #1"
```

## Пример Pull Request

Название: Добавление функции добавления книги

Описание: В ветке `feature/add-book` реализован ввод автора, названия, оценки и даты прочтения. Добавлена проверка на дубликаты книг. После проверки ветка объединяется с `main`.
